NETQUIZ WEB INSTALLATION

The Netquiz Web Installation Guide is available here: http://aide.ccdmd.qc.ca/nqw/en/node/136.

If you need help, send a copy of this file to your web hosting provider, so that the provider can prepare the server and transmit instructions to you regarding the steps required for installation.

 

INFORMATION FOR WEB HOSTING SERVICE PROVIDERS

Create a MySQL database for Netquiz Web. The MySQL user associated to this database must have the following privileges: Select, Insert, Update, Delete, Index, Create temporary tables, Execute, Create, Alter, Drop, Lock tables, References and Create routine. 

Send the following information to enable the user to complete the installation:

1) MySQL database information

  • Network address for the server containing the database
    to be entered in DB_HOST
  • Database name to be entered in DB_NAME
  • Database username to be entered in DB_USER
  • Database password to be entered in DB_PASSWORD
  • The EMAIL ADDRESS that will be used to manage the data.

 

2) FTP access information for uploading Netquiz Web to the server

  • Host
  • Username
  • Password

As needed, provide:

  • Protocol (FTP, SFTP, SSH, etc.)
  • Encryption (or coding)
  • Main folder or startup folder

 

3) The URL assigned to the Netquiz Web main folder

http://<main folder address>

 

Once the user has transferred Netquiz Web on the server, he may need your help to ensure that the web server has write permissions in the following sub-folders:

  • /ressources/journaux
  • /ressources/medias
  • /web

If you have questions or experience technical problems, contact technical support at soutien@ccdmd.qc.ca.
